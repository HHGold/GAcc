# GAcc - 現代化極簡記帳軟體

這是一款專為小家庭、情侶或個人設計的輕量級 Android 記帳應用程式。採用 **Android 原生技術 (Kotlin + Jetpack Compose)** 開發，提供流暢、具備現代質感的暗色系 (Dark Theme) 使用者體驗。

## ✨ 核心特色
- **全新極簡圖標**：採用現代感十足的「極簡錢包與硬幣」設計，並搭配具有質感的微灰背景，與桌面風格完美融合。
- **極致原生效能**：百分之百使用 **Kotlin** 與 **Jetpack Compose** 框架，確保操作流暢、無任何網頁封裝延遲。
- **針對性記帳分類**：
    - **常規支出**：餐飲、日常用品。
    - **代付管理**：支援「代付」功能，系統會自動從總支出中扣除代付金額，精準計算實際負擔。
    - **借還款追蹤**：獨立記錄借出與還款，自動計算「累計剩餘借款」（支援跨月份折抵計算），讓金流清清楚楚。
- **直覺式結算明細**：
    - **色彩標註**：代付（橘黃）、借款（紅色）、還款（青色）動態區分。
    - **聰明對帳系統**：針對「親愛兒」與「家人」提供「剩餘結清金額」與「已結清狀態標記」功能。
    - **多行顯示優化**：極致緊湊的明細佈局，小字體設計讓您在單一畫面上查看更多歷史紀錄。

- **安全本地備份 (CSV)**：
    - 支援一鍵將記帳資料雙向「匯入」與「匯出」為 `.csv` 檔案格式，轉移裝置免煩惱。

## 📱 主要畫面與功能
### 1. 記帳頁面 (✍️)
- **日期選擇器**：預設為當天日期，支援補記歷史帳目。
- **靈活分類選單**：
    - 快速選取「誰的花費」。
    - 兩排式項目分類，包含新增的「代付」與「借還款」（可細選借/還）。
- **輸入細項**：支援自定義備註（如：全聯採買、某某借錢等）。
- **高效輸入**：單行大字體金額輸入格與專屬數字鍵盤，支援「0 元」帳目儲存功能（可用於純文字備註或打平紀錄）。
- **滾動佈局**：支援垂直滾動，在小螢幕手機上操作依然輕鬆。

### 2. 結算明細 (🧾)
- **統計摘要卡片**：
    - **本月總支出**：自動加總支出並扣除代付。
    - **剩餘借款**：自動回溯結算至當月為止的歷史借還款總和，展示精準的累計剩餘欠款（跨月延續折抵）。
    - **剩餘結清金額與結清功能**：針對特定使用者（如親愛兒、家人）會額外計算「上月未結清 + 本月前支出 - 代付」，並可透過彈出式選單「整月」或「自訂區段」一鍵標記已結清，自動扣抵餘額。
- **月份與對象切換**：左右滑動月份或點擊成員，即時更新過濾清單（選擇「自己」時會貼心地隱藏不必要的對帳欄位）。
- **緊湊明細清單**：
    - 負數金額標註（借款與代付自動顯示 `- NT$`）。
    - 已結清標註（當帳目完成結算會亮起綠色 `(已結)` 提示）。
    - 快速刪除錯誤帳目。
    - 優化行高設計，一次掌握更多歷史資訊。

### 3. 設定頁面 (⚙️)
- **資料備份與還原**：點擊「匯出備份」，即可將所有記帳與結清資料完整輸出為 `.csv` 檔案備存。
- **資料匯入**：換手機或遺失資料時，透過「匯入備份」即可從 `.csv` 檔瞬間還原完整的帳戶紀錄。
- **版本資訊**：底部顯示應用程式的當前發行版本號。

## 🛠️ 開發與技術架構
- **環境要求**：Android 7.0 (API 24) 以上。
- **核心語言**：Kotlin 2.x
- **UI 引擎**：Jetpack Compose (Material 3)
- **本地儲存**：SharedPreferences + Gson (完全離線保護隱私)。
- **外部資源**：採用高解析度自定義 Mipmap Icon 組，支援各類螢幕解析度。

## 🚀 如何編譯與打包 (Build & Sign)
本專案已設定完備的數位簽名流程。

### 1. 編譯指令
打開終端機並執行以下指令來產出簽名後的 APK：

```powershell
# 移動到專案目錄
cd c:\AI\GAcc

# 清理並組建測試版 APK (Debug)
.\gradlew assembleDebug
```

### 2. 產出位置
編譯完成後，最新安裝檔會自動產出於：
- **`c:\AI\GAcc\GAcc.apk`** (根目錄便利版本)
